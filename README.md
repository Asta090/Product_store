# MERN Chakra Product Store
A Full-Stack Product Store Application

## 🛠️ Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Chakra UI v3.x

## 📁 Environment Setup
### Required Environment Variables
- `MONGO_URI`: Your MongoDB connection URI
- `PORT`: The port for the backend server (e.g., `5000`)
